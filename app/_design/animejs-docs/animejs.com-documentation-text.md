## [A collection of utility functions to help with text animations.](https://animejs.com/documentation/text/\#a-collection-of-utility-functions-to-help-with-text-animations)

All text functions are available on the `text` imported from the main `'animejs'` module:

```js
import { text } from 'animejs';

text.splitText();
text.scrambleText();




```

Or imported directly from the main `'animejs'` module:

```js
import { splitText, scrambleText } from 'animejs';




```

Or imported as a standalone module from the `'animejs/text'` [subpath](https://animejs.com/documentation/getting-started/module-imports "subpath"):

```js
import { splitText, scrambleText } from 'animejs/text';




```
