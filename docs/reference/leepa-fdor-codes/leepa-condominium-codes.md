# LeePA Condominium Code List

Source: https://www.leepa.org/Docs/Codes/Condominium_Code_List.pdf
Publisher: Lee County Property Appraiser (State of Florida)
As of (source's own "current as of" line): 12/17/2012 — "All data is current at time of printing and subject to change without notice."
Fetched: 07/20/2026

These are LeePA-internal codes used on condo unit records (extra features, extra rooms,
frontage/location/parking/quality/recreation/unit type/view). Distinct from the statewide FDOR
`DOR_UC` field — see `README.md` for the field cross-reference.
