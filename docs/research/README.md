# Research Archive

**Read this before going out to research any tool, library, or service.**

This folder stores findings from every research session. If a tool appears here, it was already evaluated. Do not re-research it. If the evaluation is stale (>3 months), note that and do a targeted update — not a full re-search.

## How to use this archive

1. Search for the tool or topic by name before opening a browser or spawning a search agent.
2. If found: read the existing entry. Use it. If it's stale, update only the changed facts.
3. If not found: do the research, then **add a new entry to the relevant session file** before closing the session.
