# Master Brain v3.0 Strategy (Arsenal)

## Overview
The Arsenal strategy transitions the Brains project from a data landfill to a formal data ledger based on a Bayesian Stack.

## The 4 Pillars
1. **SKOS Ledger**: A centralized vocabulary (`brain-vocabulary.json`) for metric disambiguation.
2. **Stack Graph Routing**: Recursive DAG traversal for deterministic confidence modeling.
3. **Deterministic Backprop**: Mathematical error attribution to identify culprit sources when confidence < 0.6.
4. **Tooling Enablers**: Integration of `dlt` (data-load-tool) and Serena for semantic code intelligence.
