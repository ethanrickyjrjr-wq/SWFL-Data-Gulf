# Step 2 — One-corridor generator end-to-end

**GATED on Step 1 (GitHub issue must be closed before starting).**
**Steps 2 + 3 ship in the same PR.**

## Target corridor

Pine Ridge Rd Naples — clean, medical-office, low confounders.

## Acceptance criteria

### Stage A — `refinery/tools/build-corridor-fact-pack.mts`

- [ ] Pure function, no network, unit-testable.
- [ ] Pulls from `corridor_profiles`, `data_lake.marketbeat_swfl`, `data_lake.bls_laus`, `data_lake.fdot_aadt_*`, `data_lake.zori_swfl`, `data_lake.fema_nfip_*`, `data_lake.lee_building_permits`.
- [ ] Output: structured JSON with `current` (value + units + `source_url`) and `important_math` (YoY delta / trailing direction, code-computed only for decision-relevant indicators).
- [ ] Missing data: `{value: null, gap_reason: "<reason>"}` — never absent keys.
- [ ] Reads prior-run `character_facts` + `character_speculative` → `prior_quarter_context`.
- [ ] Computes `fact_pack_vintage` (oldest source date).

### Stage B — `ingest/pipelines/corridor_grounded/pipeline.py`

- [ ] Matches `news_swfl` pattern.
- [ ] One `web_search_20260209` call per corridor with seed `allowed_domains`.
- [ ] Full response + all `citations[]` → `lake-tier1` bucket as NDJSON via `storage_uploader.py` + `tier1_inventory.py`.
- [ ] Re-runnable without re-spending credits.
- [ ] `--dry-run` flag (required per pipeline freshness standard).

### Stage C — `refinery/tools/synthesize-corridor-character.mts`

- [ ] Reads fact pack + Tier-1 grounded blob + prior-quarter context.
- [ ] Single model call → JSON with three keys: `facts_block` (3–5 sentences), `chart_block` (structured rows/columns or null), `speculative_block` (2–4 sentences + inline disclaimer).
- [ ] `--preview` flag: stdout only, no DB write.
- [ ] `--corridor=<name>` flag for targeted runs.

### SQL migration (same PR)

- [ ] All 6 new columns added (see epic.md for DDL). `character` column NOT overwritten.

### Manual check

- [ ] Run `--preview --corridor="Pine Ridge Rd Naples"`. Operator eyeballs output before DB write.
